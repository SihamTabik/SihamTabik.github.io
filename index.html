  • Accelerating scientific kernels on GPUs, many-core and multi-core
  • Building performance models for specific computation patterns

Software

  • Insolator: High Performance GIS-Software for Solar Irradiation Computation [Download]
    Insolator is an open source, high performance software for computing solar energy input that can be collected
    at any point of a terrain represented by a DEM (Digital Elevation Model). It is stable and can be run on distributed
    and or shared memory systems.
